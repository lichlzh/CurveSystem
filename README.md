# CurveSystem
The Assignment of ECUST Java lesson.<br/>

＜＜Java 程序设计及应用＞＞　大作业<br/>
曲线系统-目的及要求<br/>
一　目的 <br/>
1综合运用Java知识开发一个小型的软件系统：曲线系统 。<br/>
2 培养软件系统分析和设计能力。<br/>
二　要求 <br/>
1 开发一个基于GUI的动态显示任意曲线系统<br/>
2在GUI界面上演示一个自左至右移动的曲线 。<br/>
3曲线在移动过程中，同时也显示当前曲线对应的y值。<br/>
4曲线的x数据用计算机模拟。<br/>
5采用技术：GUI,图形，线程, 泛型同步技术。<br/>
6 最多2人一组<br/>
 说明：任意曲线至少4种曲线（例如sinx,cosx,x等），系统运行过程中由用户选择显示哪种曲线。<br/>

The code is from a Java beginner so it must be ugly. <br/>
